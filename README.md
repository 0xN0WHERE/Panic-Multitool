## 📦 Installation

### Option 1

1. Download the project.
2. Extract the folder.
3. Run:

```bat
setup.bat
```

This will:

- Check your Python installation
- Install all required dependencies
- Launch the tool automatically

---

### Option 2 (Manual)

Install the required packages:

```bash
pip install -r requirements.txt
```

Then start the tool:

```bash
python main.py
```
